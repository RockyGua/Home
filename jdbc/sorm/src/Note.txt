1. 建立db.properties
2. 每张表只能有一个主键。不能处理多个主键的情况。
3. po尽量使用包装类，不要使用基本数据类型。

建立表结构：
create table emp
(
  id            int(10) not null,
  empname       varchar(20) not null,
  salary        double,
  birthday      date,
  age        	int(11),
  deptId		int(10)
);

create table dept
(
  id            int(10) not null,
  dname       	varchar(30),
  address       varchar(100)
);

alter table emp add constraint PK_id primary key(id);
